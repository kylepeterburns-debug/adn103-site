How to Update This Site

This is a simple static site — no database, no login. You just edit files and upload.

  1. Add your documents: Put PDFs/Docs into the /docs folder or host them anywhere.
  2. Edit the weekly links: Open weeks.json and add entries under resources for each week.
  3. Publish: Upload the whole folder to Netlify (drag-and-drop) or GitHub Pages.

Add resource links

{
  "title": "Week 1 …",
  "resources": [
    { "title": "Week 1 Slides (PDF)", "url": "docs/week1-slides.pdf" },
    { "title": "Reading Packet", "url": "https://example.com/packet" }
  ]
}

Quick publish

  • Netlify: go to app.netlify.com → Add new site → Deploy manually and drop the folder.
  • GitHub Pages: push the folder to a repo → Settings → Pages → Source: main / root.

That’s it — once live, share your URL with students.
